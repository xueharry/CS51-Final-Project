K-Means Movie Recommender
CS51 Final Project
By Raymond Cen, Magaly Gutierrez, Michelle Ng and Wenzhe (Harry) Xue

Requirements:
Latest version of Python

Instructions:
(For Mac users)
Open up Terminal and navigate to the “code” directory. Within the “code” directory in Terminal, start the Python interpreter by entering the command “python”.

(For other users)
Open up the Python interpreter within the “code” directory. 

Type “import main” within the Python interpreter. The k-means algorithm will run until changes becomes 0, at which point there will be a series of prompts. Enter in an age, gender, occupation and desired number of movie recommendations. The Terminal will then output a  list of recommended movies from highest to lowest rated. To try again with a different set of inputs, type “reload(main)”.
